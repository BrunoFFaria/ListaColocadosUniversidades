# Lista de Colocados nas Universidades Portuguesas 2019
Antigamente era possível verificar quais os colocados assim como as respectivas notas. Infelizmente essa funcionalidade foi removida. Não sendo possível agora, ou pelo menos de uma forma fácil, verificar qual foi a nota do último colocado, assim como as notas das pessoas colocadas. Este script visa culmatar essa necessidade não só para os alunos da segunda fase, como para os professores pois podem ficar a saber de um modo rápido se o curso encheu, a nota do último colocado, assim como as dos demais, ou até mesmo fazer algumas estatísticas.
